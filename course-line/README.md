# Course-Line Report Materials

This directory contains the public course-facing materials for the CSC6133 final project report, **Latent Conditioned Relation Scoring for 3D Visual Grounding**.

The course-line framing is conservative:

- final evidence: scene-disjoint evaluation, baseline reproduction, hard-subset diagnostics, and coverage diagnostics;
- controlled evidence: Phase 4 conditioned relation-scoring ablations;
- pilot evidence: Phase 5 counterfactual and Phase 6 latent-mode runs;
- unresolved limitation: multi-anchor reasoning.

Do not merge frozen-logit diagnostics, trained Phase 4 ablations, and Phase 5/6 pilots into a single leaderboard.

## Public Files

| Path | Purpose |
|------|---------|
| `report/` | LaTeX source, figures, references, and compiled `report.pdf`. |
| `FINAL_PROJECT_EVIDENCE_MAP.md` | Evidence map used to keep claims traceable. |
| `CLAIM_BOUNDARY.md` | Conservative claim boundary for the course report. |
| `REPRODUCIBILITY_NOTES.md` | Reproducibility notes for the course-facing evidence. |

Older planning notes and presentation drafts are intentionally not part of the public course-line surface.
